# CIFAR-10 Convolutional Neural Networks

Image classification case study using deep learning models on the CIFAR-10 dataset.

## Contents

This project compares different neural network approaches for image classification, starting from a fully connected baseline and progressing towards convolutional neural networks.

The work includes model training, evaluation, regularization techniques and comparison of results.

## Topics

* Image classification
* CIFAR-10 dataset
* Fully connected neural networks
* Convolutional neural networks
* Batch normalization
* Dropout
* Data augmentation
* Model evaluation
* Confusion matrix analysis

## Project File

* `cifar10_convolutional_neural_networks.ipynb`: notebook containing the complete deep learning workflow.

## Academic Context

This project was developed as coursework for the MSc in Artificial Intelligence.

The published version preserves the original methodology, code and results. Changes are limited mainly to repository organisation, file naming and presentation.
