# Deep Learning Case Studies

A collection of selected deep learning projects developed during my MSc in Artificial Intelligence.

The repository includes practical work with dense neural networks, convolutional neural networks and recurrent neural networks applied to image classification and natural language processing tasks.

## Projects

### 01. MNIST Dense Neural Network

A first deep learning case study focused on handwritten digit classification using fully connected neural networks.

Main topics:

* Dense neural networks
* Activation functions
* Weight initialization
* Optimizers
* Regularization
* Dropout
* Early stopping
* Model evaluation

### 02. CIFAR-10 Convolutional Neural Networks

Image classification case study using convolutional neural networks on the CIFAR-10 dataset.

Main topics:

* Image classification
* Fully connected baseline model
* Convolutional neural networks
* Batch normalization
* Dropout
* Data augmentation
* Confusion matrix analysis
* Model comparison

### 03. Hate Speech RNN Comparison

Natural language processing case study focused on hate speech classification using recurrent neural network architectures.

Main topics:

* Text classification
* Tokenization
* Word embeddings
* Recurrent neural networks
* LSTM
* GRU
* Bidirectional recurrent models
* Class imbalance
* Model comparison
