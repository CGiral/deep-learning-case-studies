# MNIST Dense Neural Network

Introductory deep learning case study focused on handwritten digit classification using fully connected neural networks.

## Contents

This project uses the MNIST dataset to train and analyse dense neural networks for image classification.

The notebook explores the impact of several modelling decisions, including the number of hidden neurons, activation functions, weight initialization, optimizers, training epochs, early stopping and regularization techniques.

## Topics

* Handwritten digit classification
* MNIST dataset
* Dense neural networks
* Data normalization
* One-hot encoding
* Activation functions
* Weight initialization
* Optimizers
* Overfitting
* Early stopping
* Dropout
* L2 regularization
* Batch normalization
* Model evaluation

## Project File

* `mnist_dense_neural_network.ipynb`: notebook containing the complete deep learning workflow.
